# Client

## Steps to start

* ```yarn install --ignore-scripts```
* ```yarn start```

## Steps to test

* ```yarn install --ignore-scripts```
* ```yarn test```

## Navigation & Mock

1) Login
    * put "test" in any part of the email to log in
    ex: test@email.com
        xtest@email.com
        t@test.com
    * any password great than 3 digits will be accepted
2) Item
    * Add new items
    * delete all or one item
    * alter name of one item
